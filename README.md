## 📖 About

**CortexC** is a collection of C programming projects created to explore core computer science concepts such as:

- Data Structures & Algorithms

- File Handling
- Memory Management
- Game Development
- System Design

This repository demonstrates problem-solving skills and hands-on experience with software development using the C programming language.
